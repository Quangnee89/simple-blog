                        WHAT YOU NEED TO DO
================================================================================

1. Create Supabase Project
   Go to: https://supabase.com
   - Create new project
   - Save URL & Anon Key

2. Configure Environment
   Edit: D:\simple-blog\.env.local
   - NEXT_PUBLIC_SUPABASE_URL=<your-url>
   - NEXT_PUBLIC_SUPABASE_ANON_KEY=<your-key>

3. Setup Database
   In Supabase SQL Editor:
   - Copy SQL_SCHEMA.sql
   - Paste & RUN
   - Wait for completion

4. Optional: GitHub OAuth
   - Create GitHub OAuth App (see SETUP_GUIDE.md)
   - Get Client ID & Secret
   - Add to Supabase Auth

5. Run & Test
   npm install
   npm run dev
   Open http://localhost:3000

================================================================================
                        TROUBLESHOOTING
================================================================================

Issue: "401 Unauthorized"
→ Check .env.local URL & Key

Issue: "Can't login"
→ Email confirmation may be required
→ Check Supabase Auth settings

Issue: "Build fails"
→ Delete node_modules & .next
→ Run npm install again

Issue: "RLS Error"
→ SQL schema may not have run
→ Run SQL_SCHEMA.sql again

Issue: "Can't comment"
→ Must be logged in
→ Register & login first

For more help, see SETUP_GUIDE.md
